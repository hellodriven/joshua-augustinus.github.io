This allows us to store remote settings files to be consumed by the mobile app.

## Explanation of config values

**latestAndroidVersionCritical**: If the app version is less than this, it will show a screen forcing the user to update.

**latestAndroidVersion**: If the app version is less than this it will show a popup on each load asking the user to update. The popup can be cancelled.
